Bienvenidos a EmpresaX!

Esta es la web de EmpresaX, una empresa ficticia de desarrollo de software basado en software libre.

Notas de este diseño.

cumplimiento de estandares.
Hemos seguido los siguientes estándares XHTML 1.0 Strict y CSS 2 para visualización con la mayoria de navegadores.
Desarrollo de aplicaciones Web
Diseño sin tablas
El diseño esta realizado sin uso de tablas, permitiendo una mayor claridad y velocidad de representación.
Javascript+DOW
Hemos usado javascript+DOM para la animación de tal forma que se representa correctamente en todos los navegadores.
Dos hojas de estilo.
Se usa una hoja de estilo separada para el esquema de colores.
